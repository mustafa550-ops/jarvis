# helpers/bin — Yardımcı Binary'ler

Bu dizin, JARVIS'in ihtiyaç duyduğu yardımcı çalıştırılabilir dosyalar içindir.
Su an bos olabilir — asagidakilerden ihtiyac duyduklarınızı buraya ekleyin.

## Gerekebilecek Binary'ler

| Binary | Ne işe yarar | Nereden indirilir |
|--------|-------------|-------------------|
| `ffmpeg` / `ffprobe` | Ses dosyasi donusum / kesme | https://ffmpeg.org/download.html |
| `mpg123` | MP3 oynatici (TTS) | https://www.mpg123.de/download/ |

## Kurulum

Binary'leri bu dizine koyun veya sistem PATH'inizde oldugundan emin olun.
Windows'ta `.exe` uzantili olmalidir.

```powershell
# Ornek: ffmpeg.exe bu dizine kopyalayin
# veya
# scoop install ffmpeg   (https://scoop.sh)
```
