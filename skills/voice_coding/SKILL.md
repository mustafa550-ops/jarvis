# Voice Coding Skill

## Açıklama
Sesli komutlarla kod dosyaları oluşturma, düzenleme ve refactor etme.

## Desteklenen Komutlar
- "kod yaz <dosya> <içerik>" → Yeni dosya oluştur
- "fonksiyon ekle <dosya> <fonksiyon>" → Mevcut dosyaya fonksiyon ekle
- "import ekle <dosya> <kütüphane>" → Import ifadesi ekle
- "kod düzelt <dosya> <hata>" → Hata düzeltme önerisi
- "refactor et <dosya>" → Kod refactoring
